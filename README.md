# projeto-banco-de-dados-2025-2
Sistema de gestão de doações e voluntariado desenvolvido em SQL para a UFCA. O software conecta ONGs, doadores e voluntários sob a 3FN. Foca na implementação de relações N:N e consultas parametrizadas para relatórios de impacto social. Sem o uso de ORM, assegura a integridade via restrições no SGBD e validações na camada de aplicação.
